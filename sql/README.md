# SQL — Exercícios e Análises

Queries SQL escritas durante o estudo de SQL fundamentos e avançado,
aplicadas ao dataset real de e-commerce Olist (Kaggle).

## Dataset utilizado
**Brazilian E-Commerce Public Dataset by Olist**
kaggle.com/datasets/olistbr/brazilian-ecommerce
9 tabelas | ~100 mil pedidos | 2016–2018 | Marketplace brasileiro real

## Arquivos
| Arquivo | Conteúdo |
|---------|----------|
| 01_fundamentos.sql | 15 queries: SELECT, WHERE, GROUP BY, JOINs, subqueries |
| 02_avancado.sql | 10 queries: CTEs, window functions, UNION, subqueries correlacionadas |

## Banco de dados
PostgreSQL · Schema: olist · Interface: DBeaver Community
